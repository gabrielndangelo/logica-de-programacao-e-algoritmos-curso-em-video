Repositório de estudos do curso Lógica de Programação, do Curso em Vídeo (Gustavo Guanabara), com algoritmos escritos em Portugol usando o VisualG.

🎯 Objetivo

Este repositório não existe apenas para acumular exercícios resolvidos. O objetivo é construir uma base sólida de raciocínio lógico antes de iniciar o estudo de Java, evitando a tentação de decorar soluções prontas. Cada algoritmo aqui representa uma etapa de um processo — teoria compreendida, lógica testada e, sempre que possível, refeita sem consulta depois de um tempo.

🧠 Metodologia de estudo

O fluxo de trabalho seguido em cada conteúdo é:

I. Assistir à aula teórica completa antes de qualquer código.
II. Reproduzir o algoritmo do professor, buscando entender cada linha (não apenas copiar).
III. Resolver os exercícios propostos sozinho, permanecendo tempo suficiente travado antes de buscar ajuda — a autonomia é parte do aprendizado.
IV. Revisar exercícios antigos que geraram dificuldade, refazendo-os sem consultar a resposta anterior.

Essa metodologia prioriza retenção e raciocínio autônomo sobre velocidade de avanço no curso.

🇺🇸 Programming Logic — Portugol / VisualG (EN)

Study repository for the Programming Logic course by Curso em Vídeo (Gustavo Guanabara), with algorithms written in Portugol using VisualG.

🎯 Goal

This repository isn't just a place to pile up solved exercises. The goal is to build a solid foundation in logical reasoning before starting Java, avoiding the temptation to memorize ready-made solutions. Every algorithm here represents a step in a process — theory understood, logic tested, and, whenever possible, redone without looking at the previous answer after some time.

🧠 Study method

The workflow followed for each topic is:


Watch the full theoretical lesson before writing any code.
Reproduce the instructor's algorithm, aiming to understand each line (not just copy it).
Solve the proposed exercises independently, staying stuck long enough before seeking help — autonomy is part of the learning process.
Review past exercises that were difficult, redoing them without consulting the previous answer.


This method prioritizes retention and independent reasoning over speed of progress through the course.
