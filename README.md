# logica-de-programacao-e-algoritmos-curso-em-video
Estudos de Lógica de Programação em Portugol (VisualG), baseados no curso de Gustavo Guanabara (Curso em Vídeo). Foco em construir raciocínio lógico sólido — sem decorar soluções — como base para o aprendizado futuro. Inclui variáveis, condicionais, repetição, procedimentos, funções e vetores.
